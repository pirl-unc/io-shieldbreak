## Key critique

Anti-TIGIT (tiragolumab) + atezolizumab translational substudy (n=16). PBMC Treg reduction demonstrated but NOT tumor-compartment depletion — compartment-limited effect. Broader CITYSCAPE/SKYSCRAPER phase 3 outcomes have been mixed, which argues against interpreting this as a robust Treg-depletion therapeutic strategy.

## Per-dimension findings

### Design

Translational PD analysis of CITYSCAPE (NSCLC) and GO30103 cohorts receiving tiragolumab + atezolizumab. n=16 with paired PBMC analyses. Tumor compartment data not reported in this Treg substudy.

### Sample size

n=16 evaluable is small; exploratory PD.

### Effect magnitude calibration

Tiragolumab + atezolizumab reduced PBMC Tregs but NOT tumor Tregs (tumor-compartment data not extensively reported). Contrasts with the Fc-enhanced anti-CTLA-4 literature: anti-TIGIT's Treg effect is compartment-limited in this dataset. Subsequent CITYSCAPE and SKYSCRAPER phase 3 results have been mixed, which is contextually important.

### Missing data

Not fully assessable; standard for translational.

### Multiplicity

Multiple immune-subset analyses; multiplicity correction unclear from abstract.

### Generalizability

NSCLC-centric; clinical-efficacy context is mixed.

### Treg gating / definition

Standard CD4+FOXP3+ flow (to confirm).

### Conflict of interest & funding

Genentech/Roche industry sponsorship. Authorship includes company employees.

### Spin / framing

Not overly promotional; modest effect size reported honestly.
